# html-encoder-cli

> Encodes html thought the command line

## Install

```sh
`cd path/to/html-encoder-cli`
```

```sh
npm link
```

## Usage

```js
html-encoder $'<span>Hello</span>'
// => '&lt;span&gt;Hello&lt;/span&gt;'
```

## Dependencies

html-encoder to be placed in parent directory.
